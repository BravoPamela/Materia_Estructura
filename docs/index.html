Unidad II: Estructuras de Datos Lineales

Pilas (Stacks)

Abrir enlace
Las pilas funcionan con el principio LIFO. Útiles para recursión, backtracking y sistemas de deshacer.
  • Push (Agregar)
  • Pop (Eliminar)
  • Peek (Retornar el último elemento)
  • isEmpty (Saber si está vacía)
  • isFull (Saber si está llena)

Abrir enlace

Listas Enlazada Simple

Abrir enlace
Las listas enlazadas almacenan datos mediante nodos conectados entre sí, permitiendo inserciones y eliminaciones dinámicas.
  • Insert (Insertar)
  • Delete (Eliminar)
  • Search (Buscar)
  • Traverse (Recorrer)
  • isEmpty (Saber si está vacía)
  • Size (Tamaño de la lista)

Listas Doblemente Enlazada

Abrir enlace
Las listas doblemente enlazadas almacenan nodos con enlaces al siguiente y al anterior, permitiendo recorridos en ambos sentidos.
  • Insert (Insertar)
  • Delete (Eliminar)
  • Search (Buscar)
  • Traverse Forward (Recorrer hacia adelante)
  • Traverse Backward (Recorrer hacia atrás)
  • isEmpty (Saber si está vacía)

Listas Circular Simple

Abrir enlace
Las listas circulares simples conectan el último nodo con el primero, formando un ciclo continuo sin un final definido.
  • Insert (Insertar)
  • Delete (Eliminar)
  • Search (Buscar)
  • Traverse (Recorrer la lista)
  • isEmpty (Saber si está vacía)

Unidad IV: Estructuras No Lineales con FMX

Árboles General

Abrir enlace
Los árboles generales son estructuras de datos no lineales formadas por nodos organizados jerárquicamente, donde cada nodo puede tener varios hijos.
  • Insert (Insertar nodo)
  • Delete (Eliminar nodo)
  • Search (Buscar nodo)
  • Traverse (Recorridos: preorden, inorden y postorden)
  • isEmpty (Saber si está vacío)

Árboles Binario

Abrir enlace
El árbol binario es una estructura de datos no lineal donde cada nodo puede tener como máximo dos hijos: izquierdo y derecho.
  • Insert (Insertar nodo)
  • Delete (Eliminar nodo)
  • Search (Buscar nodo)
  • Traverse (Recorridos: preorden, inorden y postorden)
  • isEmpty (Saber si está vacío)

Grafos

Abrir enlace
Los grafos son estructuras de datos no lineales formadas por vértices y aristas, usadas para modelar relaciones y conexiones.
  • AddVertex (Agregar vértice)
  • AddEdge (Agregar arista)
  • RemoveEdge (Eliminar arista)
  • Search (Búsqueda: BFS y DFS)
  • isEmpty (Saber si está vacío)
